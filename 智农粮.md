# 三、真实业务流程建模（核心）

# 3.1 标准收粮流程

```text
车辆进站（载粮）
    ↓
第一次称重（毛重）
    ↓
现场验粮
    ↓
人工报价
    ↓
卸粮
    ↓
第二次称重（空车）
    ↓
系统自动计算净重
    ↓
系统自动计算金额
    ↓
生成收购单
    ↓
更新库存
    ↓
更新财务
    ↓
完成收购
```

---

# 3.2 业务特点

## 价格不是固定标准

价格：

不是系统自动生成。

而是：

# 人工经验报价。

参考：

```text
水分
杂质
粮食品质
市场行情
客户关系
当天行情
```

因此：

系统只负责：

```text
记录
计算
统计
保存
```

不负责：

```text
自动定价
```

# 四、单位体系与计算规则（重点）

# 4.1 地磅单位

```text
KG（公斤）
```

---

# 4.2 单价单位

```text
元 / 斤
```

---

# 4.3 净重计算

```text
净重(KG) = 毛重(KG) - 车重(KG)
```

---

# 4.4 公斤转斤

```text
斤数 = 净重(KG) × 2
```

---

# 4.5 金额计算

```text
金额 = 斤数 × 单价(元/斤)
```

# 五、工业级数据安全要求（核心）

# 5.1 所有金额与重量禁止使用 float



# 十一、数据库设计

# 11.1 users 用户表

```sql
CREATE TABLE users (
    id BIGSERIAL PRIMARY KEY,
    username VARCHAR(64) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    role VARCHAR(32) NOT NULL,
    created_at TIMESTAMP NOT NULL DEFAULT NOW()
);
```

---

# 11.2 farmers 农户表

```sql
CREATE TABLE farmers (
    id BIGSERIAL PRIMARY KEY,

    name VARCHAR(64) NOT NULL,
    phone VARCHAR(32),
    car_number VARCHAR(32),

    total_weight NUMERIC(14,2) DEFAULT 0,
    total_amount NUMERIC(14,2) DEFAULT 0,

    created_at TIMESTAMP NOT NULL DEFAULT NOW()
);
```

---

# 11.3 purchases 收购单表（核心）

```sql
CREATE TABLE purchases (
    id BIGSERIAL PRIMARY KEY,

    farmer_id BIGINT REFERENCES farmers(id),

    grain_type VARCHAR(32) NOT NULL,

    gross_weight NUMERIC(14,2) NOT NULL,

    empty_weight NUMERIC(14,2),

    net_weight NUMERIC(14,2),

    jin_weight NUMERIC(14,2),

    unit_price NUMERIC(10,4) NOT NULL,

    total_amount NUMERIC(14,2),

    status VARCHAR(32) NOT NULL,

    deleted BOOLEAN DEFAULT FALSE,

    created_at TIMESTAMP NOT NULL DEFAULT NOW(),

    updated_at TIMESTAMP NOT NULL DEFAULT NOW()
);
```

---

# 11.4 inventory 库存表

```sql
CREATE TABLE inventory (
    id BIGSERIAL PRIMARY KEY,

    grain_type VARCHAR(32) UNIQUE NOT NULL,

    total_weight NUMERIC(14,2) NOT NULL DEFAULT 0
);
```

---

# 11.5 operation_logs 操作日志表

```sql
CREATE TABLE operation_logs (
    id BIGSERIAL PRIMARY KEY,

    user_id BIGINT,

    operation_type VARCHAR(64),

    target_type VARCHAR(64),

    target_id BIGINT,

    old_data JSONB,

    new_data JSONB,

    created_at TIMESTAMP NOT NULL DEFAULT NOW()
);
```
